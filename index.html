Semantic Web

This project is about the Semantic Web, the objective was to create a Semantic application. A Semantic Web application uses data from the web, that data is in organized in RDF format. During this project we collected data from dbpedia.org, we put that together in a turtle file and used it to make an application that is supposed to help gamers choose their games.
This is a prototype.
